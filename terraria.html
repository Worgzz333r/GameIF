Description

Developer: Re-Logic
Publisher: 505 Games
Composer: Scott Lloyd Shelly
Genre: Action - adventure, Sandbox
Release date: May 16. 2011

Terraria is a 2011 action-adventure sandbox game developed by Re-Logic. The game was first released for Windows and has since been ported to other PC and console platforms.

The game features exploration, crafting, building, painting, and combat with a variety of creatures in a procedurally generated 2D world.

Gameplay

Terraria is a 2D sandbox game where players explore, build, craft, fight, and mine. The game features procedurally generated worlds, a retro pixel art style, and classic gameplay similar to Metroid and Minecraft. Players start with basic tools and an NPC guide, and gradually progress by gathering resources, crafting items, and defeating enemies.

The game offers customization options for characters and homes, as well as various difficulty modes and multiplayer co-op. Terraria also supports mods, which can add new content, features, and challenges to the game.

Development and release

Terraria's development began in 2011 for Windows, with additional platforms like consoles and mobile following in later years. The game received continuous updates, with major versions like 1.2 and 1.3 adding significant content and features.

Development transitioned between studios for console and mobile versions, with the most recent major update, Journey's End, releasing in 2020. Terraria also received support for resource packs and Steam Workshop integration in 2021, and continues to see updates focused on quality-of-life improvements.
